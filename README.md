# Mini Projects Collection

This directory contains a collection of simple JavaScript mini-projects demonstrating various functionalities like file handling, API integration, and command-line interaction.

## Projects

### 1. Todo App (`TodoApp.js`)
A simple command-line Todo application.
- **Features**: Add tasks, view tasks, and exit.
- **Usage**: Run `node TodoApp.js` and follow the on-screen prompts.

### 2. Currency Converter (`currency-converter.js`)
Converts USD to other currencies using real-time exchange rates.
- **Features**: Fetches live rates from an API, supports multiple target currencies.
- **Usage**: Run `node currency-converter.js`. Enter the amount in USD and the target currency code (e.g., EUR, PKR).
- **Dependencies**: `chalk`

### 3. File Creator (`fileCreator.js`)
A utility to create text files from the command line.
- **Features**: Prompts for file name and content, then saves the file locally.
- **Usage**: Run `node fileCreator.js`. Enter the desired file name and content.

### 4. Joke Generator (`joke-generator.js`)
Fetches and displays a random joke.
- **Features**: Retrieves jokes from an external API and displays them with styled output.
- **Usage**: Run `node joke-generator.js`.
- **Dependencies**: `chalk`

### 5. Weather App (`weather.js`)
Fetches current weather information for a specified city.
- **Features**: Displays temperature, weather description, humidity, and wind speed using the OpenWeatherMap API.
- **Usage**: Run `node weather.js`. Enter a city name when prompted.
- **Dependencies**: `chalk`

## Prerequisites

- Node.js installed.
- Internet connection for API-based projects (Currency Converter, Joke Generator, Weather App).
- Install dependencies (if not already installed):
  ```bash
  npm install chalk
  ```
